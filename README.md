Este repositório contém minhas notas de estudo e exercícios do livro *An Introduction to Statistical Learning (ISLR)*.
O objetivo é consolidar entendimento conceitual ao longo da leitura.
Projetos autorais derivados desses estudos estão organizados em outros repositórios.

O livro utilizado é de domínio público e pode ser encontrado no site oficial:
https://www.statlearning.com/
